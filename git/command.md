- 설정보기
```
git config -l'
```

- 저장소 복사
```
git clone http://....
```

- 기본 원격 저장소 설정
```
git push -u 원격저장소명
```

- 원격 저장소 등록
```
git remote add origin http://...
```

- 원격 저장소 이름 변경
```
git remote rename origin remote
```

- 원격 저장소 내려받고 rebase 하기
```
git pull --rebase
```
